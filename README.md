# Bo Li Homepage

This repository powers [libraboli.github.io](https://libraboli.github.io/).

The current redesign presents Bo Li's dual identity as an academic AI imaging researcher and vivo BlueImage Lab research leader.

## Local Development

```bash
bundle install
bundle exec jekyll serve --host 127.0.0.1 --port 4000
```

The previous static site is preserved under `legacy/original-site/`.
